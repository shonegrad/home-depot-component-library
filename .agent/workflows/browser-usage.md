# Browser Tab Management

When using browser automation:

1. **Reuse existing tabs** — Do NOT open new tabs when tabs are already open with relevant content
2. **Navigate in-place** — Use the existing tab and navigate to new URLs rather than opening fresh tabs
3. **Close unused tabs** — If tabs accumulate, close duplicates or no-longer-needed pages

This prevents browser tab sprawl and keeps the workspace clean.
