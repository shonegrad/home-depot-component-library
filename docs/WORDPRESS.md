# WordPress Integration

To feature this project in a WordPress portfolio:

1. **Link Method (Recommended)**:
   - Create a Portfolio Item in WordPress.
   - Add a "Launch Project" button linking to the GitHub Pages URL: `https://shonegrad.github.io/home-depot-component-library/`.
   - Embed screenshots or a video walkthrough in the post content.

2. **Iframe Method**:
   - Use an iframe block:

     ```html
     <iframe src="https://shonegrad.github.io/home-depot-component-library/" width="100%" height="800px" style="border:none;"></iframe>
     ```

   - *Note*: Ensure the height is sufficient for the content.
